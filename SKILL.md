## 一、核心原则

### 段落管理
- **一段一个核心事件**。多人交互（A拔剑→B骤缩→A前进）若属因果链上连续动作，算一件事
- **时长上限**：单段最长15秒。单人静态情绪3-5秒，简单动作5-8秒，双人对话8-12秒，复杂动作链12-15秒。超时必拆
- **每段独立完整**，禁止写"同上""同场景""承接上段""继承上一段"。AI视频工具无记忆

### 拆段决策公式
S'' = 0.4×P + 0.3×E + 0.3×A + L
P=剧情密度，E=情绪强度，A=动作复杂度，L=长镜头触发因子(0~0.3)

| S'' 范围 | 决策 |
|-----------|------|
| ≥1.0 | 长镜头单段(15秒不切镜) |
| 0.7~1.0 | 拆3段+ |
| 0.4~0.7 | 拆2段 |
| <0.4 | 单段 |

### 段落衔接
- **换景别用动作接**：上段动作未完成，下段用不同景别接上
- **人物反应镜头**：用另一角色反应作为下段开头
- **空镜过渡**：用环境空镜作为情绪节奏过渡
- 场景首段不需继承；已转场不需继承

### 台词处理
- **100%原样嵌入动作**，严禁改写、概括、精简
- 台词紧跟动作描写，不孤立存在
- 单段台词不超过40字
- 超长台词顺延拆分至下一段（[1]上半句、[2]下半句），绝不删减

### 旁白处理
- **无效旁白改画面**：重复动作说明、低信息量转场、已能画面表达的动作 → 改成画面动作
- **有效旁白保留不动**：推进剧情、交代关键信息、补充人物心理、解释时间地点

### 动作原则
- 必须有动机：写"他为什么这么做"，不写"他做了什么"
- 人物要有目标、有情绪、有反应链

### 语法原则
- 使用"主语+动词+状语"短句结构，避免复杂从句
- 示例：【旗帜猛烈飘扬】【雨滴缓慢溅落】

### 同框人物控制
- 2人：全清晰
- 3人：主角清晰、次角虚化/背影
- 4人以上：切分轮切，每次只呈现2人关系

### 角色描述
- 无图片引用时：首次描述写全身视觉特征（颜色、衣服、发型），后续可简写
- 有图片引用时：省略外观描述，仅写动作、表情、微表情（见下方图片引用规范-着装禁用）
- 示例（无图片）：【红色运动服黑长发女孩戴粉色兔耳毛线帽】→ 后续简写为【红衣兔耳帽女孩】

### 图片引用规范（Seedance 2.0 角色参考）

角色绑定格式：`角色名=角色名`（自动从剧本场景头提取，右侧为占位符，用户后期替换为实际图片引用）
场景绑定格式：`场景名=场景名`（同上）

引用声明位置：在每段开头的"出场角色"字段中统一声明该段出现的角色与图片绑定关系。

**规则**：
- 角色绑定供身份一致性参考，首次角色视觉描述仍需完整
- 同一角色在多段中出现时，每段"出场角色"字段均需重新声明角色绑定（段间无记忆）
- 场景在首段声明一次即可，后续段落若场景不变可省略
- 单段最多声明2个角色绑定
- **着装禁用**：已声明角色绑定的角色，禁止在画面描述中写任何着装/发型/颜色等外观描述。绑定已锁定视觉身份，文字只描述动作、表情、微表情、身体反应。首次出场也不例外

示例声明：
```
出场角色：顾阳=顾阳、孙鹏=孙鹏，单元楼下=单元楼下
```

### 剧本忠实度
- 剧本事实零删减：角色、动作、道具、受伤、情绪、台词100%映射
- 动态精简：动作复杂时精简环境渲染，把算力留给动作
- 无损拆分：信息量过大时平摊到[1][2][3]连续切镜中

## 二、提示词四段结构

每段提示词按以下顺序输出，映射到输出模板的对应字段：

| 结构层 | 写入模板字段 | 内容 |
|--------|------------|------|
| ① 光影基调 | 场景与光线 | 景别 + 运镜 + 环境 + 光线基调（含K值+照度三层） |
| ② 动作叙事 | 画面描述 | 动作链 + 互动细节，因果顺序用【→】连接 |
| ③ 情绪收束 | 画面描述末尾 | 收束镜头 + 情绪落点（微弱呼吸、微距晃动延续情绪） |
| ④ 质感补充 | 质感关键词 + 约束词 | 中文质感描述 + 英文关键词 + 负面提示 |

注：② 和 ③ 合并写入同一"画面描述"字段（②为动作主体，③为结尾收束），不分成两个字段。

## 三、叙事流写法

像写小说段落一样写提示词。一段=一个连续动作/情绪。

### 关键规则
- **大运动不写**（推镜/环绕/拉远），模型自动处理
- **只写微运动**：手持抖动/画面震动/手持摄影风格
- **时间顺序写**：按动作发生先后
- **动作链用→连接**：一个动作引发下一个
- **光线融入环境**：写"环境+光源"，不写"光线"
- **表情写进动作**：写"喉结不自觉滚动，手指微颤"，不写"他很紧张"
- **台词跟动作**：台词紧跟动作描写

### OS（内心独白）
- 动作描写后标注 `OS：独白内容`
- OS画面每秒至少1个显著细节变化
- 示例：她嘴角微微上扬，OS：终于等到这一天了。手指不自觉握紧拳头，指节泛白。

### 环境音效与OS隔离
- 环境音效（风声、犬吠、雨声、脚步回声等）必须写在动作描写或环境描述中，严禁紧贴OS标注之后
- OS仅承载旁白/独白文本。环境音效属于场景描写，两者必须用动作或光线描述隔开
- 错误：`OS：直到第八天。远处零星犬吠。`（犬吠会被模型当作旁白朗读）
- 正确：`远处零星犬吠，夜风穿过灌木。OS：直到第八天。`（音效在OS前，属于场景）

### OS语速约束

中文旁白自然语速为3-4字/秒。OS字数 ÷ 3.5 = 所需最少秒数（向上取整）。

- OS内容必须能在所属时间块内以自然语速读完。若超时：扩展该时间块时长，或将OS拆分为两句分属两个连续时间块
- 示例：18字OS → 需要≥5秒；25字OS → 需要≥7秒
- 禁止在过短时间块内塞入长OS导致语速崩坏

### OS与对白的声学差异

OS 和对白是两种本质不同的声音。不是"有没有角色名"的区别——是音色、节奏、呼吸模式的根本差异。Seedance 靠声学参数区分两者，不靠文学隐喻。

#### 差异对照表

| 声学参数 | OS（回望、情绪已消化） | 对白（即时、情绪正在发生） |
|---------|---------------------|--------------------------|
| 音调起伏 | 平坦、极少变化 | 随情绪波动，有升有降 |
| 节奏 | 均匀、可控、可压缩 | 不规则——被情绪打断，呼吸即节奏 |
| 爆破音 | 正常/松弛 | 愤怒时咬重，心虚时弱化，克制时刻意压低 |
| 换气 | 规律、无声或极轻 | 情绪驱动——粗重/急促/被压住/叹息式呼出 |
| 停顿 | 语法停顿，可预测 | 情绪停顿——在不可预测的位置沉默 |
| 声带张力 | 松弛（事件已结束） | 紧绷（正在承受） |

**一行公式**：
> **OS = 音调平坦 + 节奏均匀 + 换气规律 → 听感："已经过去了"**
> **对白 = 音调波动 + 节奏不规则 + 换气情绪化 → 听感："正在发生"**

#### 心理→声学翻译表

Seedance 只能处理可量化的声学参数。所有心理描述必须翻译为声学指令。判断标准：你能不能张嘴模仿出来？能 → Seedance 能用。不能 → 换写法。

| 心理状态 | 声学参数 | 提示词写法 |
|---------|---------|-----------|
| 平静、抽离、已接受 | 语速规律、音调平坦、句尾不升不降、换气轻 | `OS：（语速规律无起伏，音调平坦，句尾自然下沉）` |
| 冰冷、克制、审判 | 音调低沉平稳、每字间距精确均匀、零情感波动、换气无声 | `（音调低沉，每字间距如节拍器，无任何重音，换气无声）` |
| 愤怒但强行压制 | 爆破音加重、咬字用力、音量和音调被压在正常范围、换气粗重 | `（爆破音咬得极重但音量被压低，每句之间有粗重鼻息）` |
| 疲惫、释然 | 语速偏慢、句尾自然下落、换气长而轻、声带松弛 | `（语速缓慢，句尾下沉，换气为长而轻的叹息式呼出）` |
| 崩溃边缘 | 声带颤动出现微颤音、音调不稳、句子被短促换气打断 | `（声音不可控微颤，句子被急促换气打断，末字几乎失声）` |
| 克制到破裂 | 前半句声学参数如"冰冷"，后半句某一个字突然失控——爆破音加重、音调骤变、换气紊乱 | `（前句平稳如节拍器，"七天"二字爆破音骤重，换气从无声变为粗重，旋即恢复平稳）` |

#### 提示词写法

OS 和对白在提示词中必须携带独立的声学指令，不能共用同一个修饰语：

```
错误（OS 和对白无法区分）：
OS：我再次开口，声音里不带任何情绪。
顾阳：（冰冷地）你知道吗？
→ Seedance 可能将两者读成同一音色

正确（两种声学参数各自独立）：
OS：（语速规律无起伏，音调平坦，句尾自然下沉）我再次开口，声音里不带任何情绪。
顾阳：（音调低沉，每字间距精确如节拍器，爆破音咬重但音量压低）你知道吗？
```

## 四、光影与氛围

### 光影底层逻辑

光的存在只有一个目的：**让观众看到需要看到的，同时隐藏不需要看到的。**所有技术选择（色温、方向、光比、强度）都是这个原则的技术实现。

#### 原则一：光必须有动机

每一束光必须可溯源。光源必须在场景中有物理存在——窗、路灯、台灯、手机屏幕、门缝漏光。即使是艺术化夸张，也要先建立现实锚点再偏离。没有动机的光 = 观众潜意识出戏。

写法：不写"面部被5000K柔光覆盖"，写"5000K台灯从左侧漫反射至面部"。光不是抽象的K值——是一盏具体的灯，在一个具体的位置，产生了一个具体的光影事件。

#### 原则二：光比 = 情绪

光比（亮面与暗面的亮度差）是情绪基调最根本的决定因素，先于色温选择。

| 光比 | 情绪 | 用法 |
|------|------|------|
| 高光比（亮暗差异大） | 冲突、压迫、戏剧化、审判 | 单光源侧逆，暗面几乎无补光 |
| 中光比 | 克制、真实、叙事推进 | 主光+柔光补光，暗部保留细节 |
| 低光比（亮暗接近） | 柔和、安全、日常、温情 | 多方向柔光均匀覆盖，阴影极浅 |

光比选择在写任何K值之前决定。不是"选什么色温"，而是"这场戏需要多少阴影"。

#### 原则三：光随叙事演进

整场戏的光不是一成不变的。角色在空间里移动 → 光在他身上移动。他站起来 → 进入直射光。他蹲下 → 沉入暗部。他走到窗前 → 光的角度和强度改变。光的动态变化本身就是叙事。

写法：不写"孙鹏面部被柔光覆盖"，写"孙鹏从暗区抬升进入台灯直射范围——面部光照骤然变化"。

#### 原则四：光与影是同一系统

主光、辅光、轮廓光不是三个独立选择——是同一束光在不同表面上的行为。光打在脸上 → 亮面。打不到 → 暗部。擦过边缘 → 轮廓。它们是一束光的三个产物。

写法：不写"主光5600K+辅光5000K=阶调方案"，写"5600K台灯从左上方直射——左脸亮面层次分明，右脸自然沉入暗部仍保留皮肤纹理，颧骨边缘被同一束光擦出冷白轮廓线"。

### 光线公式（技术实现）

将上述底层逻辑转化为具体描述：暗环境 + 一束有名字的光源（方向+色温K值+光质） = 亮面效果 + 暗部效果 + 轮廓效果 + 光比定调

### 色温参考（硬底线：4800K）

禁止使用低于4800K的光源描述。黄光/暖光在AI视频中易产生油蜡质感皮肤，所有光线必须保持在4800K以上的中性白至冷白区间。

| K值范围 | 光质 | 氛围 | 适用 |
|----------|------|------|------|
| 4800-5200K | 中性白 | 真实、清透、无偏色 | 面部柔光补光（基础档） |
| 5200-5600K | 微冷白 | 清醒、克制、干净 | 室内自然光、柔和主光 |
| 5600-6000K | 冷白 | 锐利、冷感、轮廓分明 | 路灯侧逆、月光、硬质主光 |
| 6000-7000K | 冷蓝调 | 孤寂、压迫（慎用） | 极夜、深空氛围 |

### 光源分层公式

对比不依赖蓝vs黄，而依赖冷白与中性白的阶调落差。全部光源保持在4800K以上。

主光源K值（方向+质感） + 辅光/补光K值（人脸漫反射） = 阶调控制方案。

示例：
- 5600K冷白路灯侧逆 + 5000K中性柔光面部 = 克制夜景（冷锐轮廓 × 清透肤色，零黄光）
- 5500K微冷窗光正面 + 5200K中性白补光 = 柔和内景（微寒主调 × 干净肤色）
- 6000K冷白顶光 + 5000K中性柔光面部 = 都市室内（冷锐上部 × 清透正面）

### 照度控制

照度通过三层叠加描述，不依赖绝对数值：

**第一层：光源强度词**
| 强度 | 词汇 |
|------|------|
| 极弱 | 微弱余光、仅剩微光、暗部边缘 |
| 弱 | 柔和、低强度、轻扫 |
| 中 | 均匀覆盖、正常照度 |
| 强 | 倾泻、直射、明亮 |
| 极强 | 刺眼、强烈、过曝边缘 |

**第二层：衰减暗示**
通过距离+遮挡暗示照度衰减：远处路灯仅剩微光 / 近距冷白灯管直射 / 门前廊灯被屋檐遮挡仅在地上投下微弱光圈

**第三层：曝光边界**
明确曝光状态：正常曝光层次分明 / 略欠曝 / 深度欠曝的暗区 / 高光细节即将溢出的过曝边缘

**三层组合示例**：
- 极暗：面部仅被远处路灯的微弱余光照亮，处于略欠曝的暗部边缘
- 正常：近处柔和冷白光均匀覆盖面部，正常曝光层次分明
- 强对比：近距冷白灯管直射左脸，右脸因无补光沉入深度欠曝的暗区

### 曝光控制

控制面部补光强度，避免高光溢出和过曝。关键用法：
- 补光描述中加入"低强度""柔和""控制光比"等强度限定词
- 面部特写/近景必须标注"保留高光细节不溢出"或"面部细节清晰无过曝"
- 负面词追加：面部过曝、脸部高光溢出
- 禁止任何暖黄光/黄光描述。AI中黄光易触发油蜡质感皮肤，全部光源保持在4800K以上

### 锥形光陷阱防护

AI容易将"顶光+直射"解析为舞台追光灯/锥形光束。以下写法坚决避免：

| 触发词 | AI解析结果 | 替换为 |
|--------|-----------|--------|
| 顶光从正上方直射 | 舞台追光灯、锥形单束光 | 声控灯在门上方，光线向四周均匀扩散覆盖门前区域 |
| X在光里，Y在阴影里 | 光区/暗区二元切割→锥形边界 | X站立的身躯在灯下自然投出影子，Y蹲在他的影子边缘 |
| 高光比（单光源时） | 锐化明暗切割线→锥形轮廓 | 中高光比，亮部与暗部过渡柔和自然不形成硬切边缘 |
| 一束光/一道光 | 锥形/线形光束 | 一盏灯的光线/灯光扩散覆盖 |

**光源铁律**：永远写"一盏实在的灯照亮一片区域"，不写"一束光从某个方向打下来"。区域内的物体和人物因身体遮挡、距离衰减、角度不同各自产生自然明暗——而不是被锥形光选择性照亮。

公式：`灯名+位置+扩散方式+照亮区域→区域内自然明暗`

### 死黑与过曝防护

AI视频模型对极端影调的诠释不稳定："暗处"可能变成纯黑像素（死黑），"亮光"可能变成纯白块（过曝）。所有光影描述必须加保留细节的定量锚点。

**暗部防护**：任何写到阴影/暗处/暗区的地方，必须同步声明暗部仍有可辨识细节。

| 禁用（触发死黑风险） | 替换为 |
|---------------------|--------|
| 面部沉入暗处 | 面部沉入暗部——仍可辨识颧骨轮廓与皮肤纹理，冷调边缘光微勾勒面部结构 |
| 完全隐在暗处 | 处于暗部，极浅的冷调边缘光在鼻梁和下颌处保留微弱立体感 |
| 半张脸沉入暗处 | 半张脸处于暗部——暗侧并非纯黑，颧骨与眼窝结构在微弱漫反射中仍可辨识 |
| 阴影吞没 | 阴影覆盖——最深处仍保留皮肤质感，不出现纯黑像素 |

**亮部防护**：任何写到光照/亮面/轮廓光的地方，必须同步声明高光有层次不溢出。

| 禁用（触发过曝风险） | 替换为 |
|---------------------|--------|
| 冷白光照亮面部 | 冷白光覆盖面部——高光区域保留皮肤纹理，亮面层次分明不溢出纯白 |
| 强烈轮廓光 | 明亮轮廓光——边缘高光保留渐变过渡，不形成纯白切割线 |
| 光线打在脸上 | 光线漫反射至面部——亮部到暗部形成柔和过渡，高光细节完整 |

**通用锚定词**（可插入任何光影描述末尾）：暗部保留细节、高光有层次、亮暗过渡柔和、不出现死黑、不出现纯白过曝。

### 可用词
逆光、轮廓光、发丝光、面部柔光漫反射、侧逆光、明暗交界、光斑、阴影
冷白调、中性白、微冷白、冷阶对比、阶调分层、色温层次、精确K值、肤色微反差、清透皮肤纹理、清爽肤色

### 禁用词

### 景别对应
| 景别 | 光线处理 | 色温策略 |
|------|---------|---------|
| 大全景/远景 | 整体色调氛围 | 定环境主色温基调 |

### 电影级质感强化
- 前景遮挡：特写/近景引入焦外飘雪、模糊木梁边缘
- 极浅景深、微表情光影（伦勃朗光/眼神光/眼球微颤动）

## 五、运镜与分镜决策

分镜选择不是从菜单里点菜。每一次景别、角度、运镜的决定，都回答同一个问题：**摄影机此刻的立场是什么？**

### 先决：摄影机立场

在写任何分镜之前，先回答：这场戏的摄影机是谁？
- **审判见证者**：不站任何角色，但忠实记录权力差——用角度高低翻译谁压制谁
- **角色主观（POV）**：摄影机就是角色的眼睛——画面中绝不能出现该角色本人
- **情绪共生者**：摄影机与某个角色的情绪同步呼吸——用手持微晃翻译内心不稳定

同一场戏内，立场可以切换，但每次切换必须有叙事动机——不能仅仅因为"该换景别了"。

### 分镜三优先决策框架

每次选择景别/角度/运镜，按以下优先级依次决定：

#### 第一优先：权力关系 → 角度

角度是权力差的视觉翻译。这是最重要的决定，因为它决定了观众"站在哪里看"。

| 权力状态 | 角度 | 效果 |
|---------|------|------|
| A 压制 B | 微仰 A（从B的POV或客观低于A的位置）、微俯 B | A 在画面中占据上方，B 被压在下方 |
| 平等对话 | 平视 | 两人高度、画幅占比相等 |
| A 被击垮/孤立 | 俯角从 A 视角缓拉至全景 | A 在画面中越来越小 |

**这场戏的贯穿原则**：如果某人站着俯视、另一人蹲着仰视——这个角度差必须焊死在视觉系统里，不是偶尔来一下，是每一个镜头都携带。

#### 第二优先：情绪距离 → 景别

景别 = 观众与角色的情感距离。不是"该换景别了"的机械轮换。

| 情绪状态 | 景别选择 | 原因 |
|---------|---------|------|
| 冰冷克制、审判、拒绝投入 | 中景 → 中近景 | 距离感 = 情感上的不投入。观众被挡在安全距离外 |
| 情绪裂缝、破绽、内心松动 | 近景 → 特写 | 距离消失 = 观众被拉进角色的内部，无法再保持距离 |
| 崩溃、被压垮、彻底孤立 | 全景 | 距离骤然拉远 = 角色的渺小、孤独、被世界抛弃 |
| 建立空间、环境压迫感 | 远景/全景 | 环境大于人 = 人被空间吞没 |

**关键规律**：景别推进的方向 = 情绪是在靠近还是在远离。缓推 = 把观众拉进去（情绪聚焦），缓拉 = 把观众推出来（情绪抽离）。

#### 第三优先：呼吸节奏 → 运镜

运镜 = 摄影机的呼吸。稳定 = 控制，晃动 = 失控。

| 叙事节奏 | 运镜 | 何时使用 |
|---------|------|---------|
| 稳定压迫、不可动摇 | 固定 | 角色在克制、在审判、在掌控。摄影机如磐石，角色在框内被压住 |
| 内心不稳定、紧张、冲突 | 轻微手持 | 角色情绪出现裂缝。摄影机与角色的身体同步呼吸、同步失稳 |
| 真相逼近、不可抗拒的情绪高潮 | 极缓推 | 一种无法阻止的靠近感。观众被推着向前，没有退路 |
| 情绪抽离、留白、结束 | 缓拉 | 观众被从情感中心推开。常用于收束和转场 |
| 入场、追逐、逼近 | 跟拍 | 角色在移动，摄影机与之保持固定距离 |
| 空间揭示 | 摇镜/平移 | 逐步展开场景，建立环境认知 |
| 强关系张力、爆点前夕 | 环绕 | 极度克制使用——仅用于两人对峙的紧张巅峰 |

**手持原则**：若某段情绪以克制为主 → 固定。只有当角色出现明确的情绪裂缝（身体失控——摇头、颤抖、呼吸紊乱）时，摄影机才同步失控（切入轻微手持）。手持不是默认值，是情绪失控的信号。

### 微运动（需要写）

大运动（推镜/环绕/拉远）不写，模型自动处理。只写微运动：手持抖动、画面震动、手持摄影风格。

### 景别推进参考模式

以下模式是常见叙事节奏的参考，而非机械配方。实际分镜应以三优先框架为准。

- **环境建立**：远景/全景 → 中景 → 中近景 → 近景 → 特写
- **情绪推进**：中景 → 中近景 → 近景 → 特写
- **对话推进**：双人中景 → OTS过肩 → 单人近景 → 反应特写
- **动作推进**：全景/中景 → 跟拍中景 → 动作特写 → 结果近景
- **爆点推进**：稳定中近景 → 快速推近 → 特写钉情绪

### 时间分块运镜

- 单镜头段：每段时间块写相同运镜
- 多镜头段（切镜）：每段时间块写不同运镜，切镜即换运镜

### 主观镜头（POV）

POV = 摄影机就是角色的眼睛。

**规则**：
- 画面中绝对不能出现该角色的身体、衣物、剪影。只能看到该角色所看到的
- 描述视角所见而非角色本身：写"孙鹏的脸被阴影吞没"，不写"顾阳的剪影压在孙鹏脸上"
- 运镜统一：同一段POV内运镜类型贯穿不变，手持程度与角色呼吸同步
- 画外音（换气声、脚步声、对话）可以来自POV角色，从画面上方/后方传来

### 镜头连续性

- **单镜头连续段**：同一POV或同一机位内，禁止切镜、禁止运镜类型来回切换（如微晃→固定→微晃）。推进/拉远必须是渐进的（"极缓慢推近"），不能在某秒突然跳跃切换
- **多镜头切镜段**：切镜 = 摄影机位切换，用【切镜】标注，切镜后运镜可以不同
- **手持统一**：若声称"手持贯穿全程"，则所有时间块的运镜描述必须一致，不能出现"固定"字样

## 六、人物刻画

### 底层逻辑

身体是一个统一系统。情绪不是面部表情 + 手部动作 + 呼吸变化的叠加——情绪是神经信号在身体各部位的传播，有方向、有先后、有因果。

#### 原则一：身体统一系统

一条刺激进入角色后，它在身体各部位的传播顺序是固定的：

**刺激源 → 呼吸（最快，最不可控） → 眼部（其次） → 嘴部/喉部 → 手部（潜意识小动作） → 身体重心/姿势（最慢，半可控）**

不是同时发生。不是分别描述。是一个完整的神经事件在身体表面的逐次显现。

错误写法（零件拼装）：`他眼神下坠，嘴唇拧住，手指揪紧布料，呼吸变浅。`

正确写法（因果链）：`他听到"喂了狗"——呼吸在喉咙里卡了半拍。眼神短暂下坠。嘴唇拧住又松开。呼吸恢复时手指下意识揪紧布料。`

#### 原则二：动作节拍三段式

每个有意义的动作都有三个节拍。只写执行 = 机器人。

| 节拍 | 生理内容 | 示例（孙鹏肩膀震颤） |
|------|---------|---------------------|
| 预备 | 肌肉收紧、呼吸变化、身体微调 | 胸腔骤然僵住，呼吸在喉咙里卡了半拍 |
| 执行 | 动作本身 | 肩膀剧烈震颤了一下 |
| 余波 | 肌肉释放、呼吸恢复、次级效应、永久性体态变化 | 震颤过后肩膀没有恢复原位——向内又缩了一寸，手指掐进大腿的力度比之前更重 |

余波是最容易被忽略但最能制造真实感的节拍。动作结束 ≠ 事件结束。身体需要时间回到基线——有时它回不去。

#### 原则三：微表情的因果链

微表情必须有前置刺激，且刺激必须在画面中可见或刚刚发生。

不是：`他喉结滚动。`

而是：`因为他听到"你爸"两个字——喉结急促地滚动了一下。`

因果箭头用【因为……——】或【→】连接。没有前置刺激的微表情 = 悬浮表演，观众不知道角色在对什么做出反应。

#### 原则四：单一焦点

同一时刻只写一个最重要的身体信号。不是每个时刻都要覆盖全身——是"此刻他最诚实的身体部位在做什么"。其他部位可以沉默。留白比堆砌更真实。

判断标准：如果去掉这个描写，观众还能看懂角色的情绪吗？如果还能——这个描写是冗余的。

### 肌肉链力学与抗僵原则

微表情的本质是面部肌肉链的接力收缩。了解哪块肌肉先动、带动哪块、产生什么表面效果，才能写出 AI 能精准呈现且不僵硬的表情。

#### 关键表情的肌肉链传播顺序

**愤怒/克制愤怒**：
```
第一波（不可控，0.1-0.3s）：皱眉肌+降眉间肌 → 眉头下压，鼻根横向收紧
第二波（半可控，0.3-0.6s）：咬肌收紧 → 后槽牙处皮肤向外凸起，下颌角轮廓变硬
第三波（可控但已泄露，0.6-1.0s）：胸锁乳突肌 → 喉结滚动时颈部两侧肌腱短暂凸显
```
克制愤怒 = 上述肌肉链全部启动，但被对抗肌群反向锁死。表面结果是皮肤下的微凸起、微震颤——两股力量在对抗，谁也赢不了。

**恐惧/心虚**：
```
第一波：额肌 → 眉毛上抬，额头横纹
第二波：眼轮匝肌 → 眼睑紧张，眨眼频率骤增
第三波：颈阔肌 → 颈部皮肤向下拉紧
第四波：胸锁乳突肌 → 喉结急促上下滚动
```

**蔑视/冷笑**：
```
非对称启动（核心特征）：
  单侧颧小肌+提上唇肌 → 同侧嘴角上挑，同侧鼻唇沟加深
  对侧完全不动 → 半张脸在笑，半张脸是冰
  眼轮匝肌不参与 → 眼睛不笑（冰冷感的解剖来源）
```

**崩溃前克制**：
```
眼轮匝肌 → 眼眶泛红（毛细血管扩张），眼睑强行撑住
降口角肌 vs 颏肌对抗 → 嘴角被下拉，下巴反向顶住 → 嘴唇在颤抖中维持不稳定直线
甲咽肌收缩 → 声带闭合 → 声音发咽、被压住
```

#### 表面效果翻译公式

AI 不懂肌肉名。写皮肤表面的可见变化：

`肌肉事件 → 皮肤表面的位移、凸起、褶皱、光影变化`

| 肌肉事件 | 错误写法（AI不懂） | 正确写法（AI能呈现） |
|---------|------------------|--------------------|
| 咬肌收紧 | masseter contracts | 后槽牙处皮肤向外凸起，咬合肌如钢缆般撑开下颌角 |
| 皱眉肌+降眉间肌 | corrugator + procerus | 眉心皮肤出现纵向浅褶，鼻根处横向收紧 |
| 口轮匝肌张力增加 | orbicularis oris tightens | 嘴唇向牙面压平变薄，唇线变得锋利 |
| 胸锁乳突肌紧张 | sternocleidomastoid | 喉结滚动时颈部两侧肌腱短暂凸显，随即隐回皮下 |
| 颧小肌单侧收缩 | zygomaticus minor unilateral | 左侧嘴角向上微微一挑，右侧纹丝不动 |
| 降口角肌 vs 颏肌对抗 | depressor vs mentalis | 嘴角被向下拉，下巴肌肉反向顶住——嘴唇在颤抖中维持不稳定直线 |
| 眼轮匝肌不参与 | orbicularis oculi absent | 嘴角在上扬，眼周皮肤完全不动——笑意没有到达眼睛 |

#### 三条抗僵原则

**原则一：非对称是默认值。**人脸天然不对称。至少 50% 的面部描写应携带非对称信息。AI 从对称训练数据中学到的表情是镜面反射——一旦指定非对称，模型被迫脱离模板，动态必须真实计算。

```
僵硬：他嘴角上扬
抗僵：左侧嘴角先微微一挑，右侧慢了半拍才跟上——幅度也更小
```

**原则二：写过渡态，不写终点。**肌肉收缩不是瞬时的——有加速度、微颤、过冲然后回弹。AI 擅长"起点→终点"线性插值。只给终点 = 僵硬。描述过渡态中的扰动 = 强迫 AI 生成更复杂的运动曲线。

```
僵硬：他眉头下压
抗僵：眉心皮肤先极轻微地抽动了一下，然后才缓慢地向内收紧——皱眉肌在收缩过程中出现了一次微不可察的颤抖，旋即稳定
```

**原则三：写对抗肌群，不写单肌肉。**克制表情 = 表达肌群启动 + 对抗肌群同步锁死。两股力量对冲 → 表面微凸起、微震颤、不稳定平衡。AI 无法生成"干净"的表情——在两个冲突方向之间计算折中，自动产生真实感。

```
僵硬：他嘴唇颤抖
抗僵：嘴唇剧烈颤抖——但下颌咬合肌死死压住，试图让嘴唇恢复直线。结果是嘴唇在颤抖和僵直之间来回震荡，始终没有稳定在任何一个状态
```

---

### 表情语言：用肌肉动作替代情绪标签

情绪 = 眼神方向 + 眉眼状态 + 嘴唇状态 + 呼吸节奏 + 手部动作 + 身体重心 + 语气变化。永远不写"他很愤怒"——写咬合肌如何凸起、呼吸如何变重、眼神如何锁定。

### 基础表情参考

- 愤怒：咬紧牙关，下颚绷紧，眉头下压
- 震惊：瞳孔骤然收缩，嘴巴不自觉微张
- 恐惧：眼神慌乱闪躲，鼻翼翘动
- 冷笑：嘴角向一侧上扬，眼神冰冷
- 心虚：眼神避开，眨眼频率增加，嘴唇抿住又松开，重心后移
- 杀意：眼神完全锁定目标，眨眼减少，眉眼压低，呼吸平稳到异常，身体微微前倾
- 崩溃前克制：眼眶泛红但不落泪，呼吸被压住，喉结滚动，嘴唇轻颤

### 工具包（非检查清单）

以下工具用于在写微表情时寻找具体的身体信号——不是每项都要出现，而是当角色此刻最诚实的部位是眼/手/呼吸时，从中选取最精确的描述。

#### 七个反僵尸维度（工具包，勿逐项套用）

| 维度 | 示例 |
|------|------|
| 视线流转 | 视线从...移动到... |
| 非对称微表情 | 左侧嘴角微扬，右侧纹丝不动 |
| 潜意识生理反馈 | 眨眼、喉结滚动、呼吸起伏、手指微颤 |
| 身体重心微动 | 重心微后移、脚尖转向门口、指节泛白 |
| 环境互动 | 手指摩擦桌面刀痕、鞋底磨过地板水渍 |
| 局部光影 | 半张脸在暗处、半张脸被窗光打亮 |
| 表演指导 | 谨慎、震惊、吞吞吐吐、一字一顿、面露冰冷的微笑 |

#### 情绪递进链（整场戏的宏观弧线）

隐藏情绪 → 情绪泄露 → 情绪对抗 → 情绪爆发 → 情绪收束

#### 微表情词库

- **眼部**：下意识地眨了下眼、眼睫低垂遮住了眼底情绪、眼尾微微泛红、瞳孔骤然收缩/放大
- **眉部**：眉心微不可察地皱了一下、单侧眉毛微微挑起
- **嘴部**：双唇抿成一条细线、嘴角几不可察地抽动、无意识舔了下干燥的嘴唇
- **面部**：下颌微收时咬合肌微微凸显、耳尖不易察觉地泛红

#### 生活化小动作词库

- **手部**：手指无意识敲击桌面、用拇指反复摩擦食指关节、无意识摩挲杯沿
- **整理动作**：整理袖口、拨正衣领、下意识拉平衣摆
- **头部**：无意识拨弄耳边碎发、偏头时耳坠轻轻晃动
- **身体**：脚尖不自觉转向门口、无意识地将重心从左脚移到右脚
- **面部**：无意识舔了下干燥的嘴唇
