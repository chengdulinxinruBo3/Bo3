详细能力

  • 段落管理 — 智能拆段算法 S''，单段最长 15 秒，超时自动拆分
  • 段落衔接 — 换景别用动作接 / 人物反应镜头 / 空镜情绪过渡
  • 旁白智能处理 — 无效旁白改画面，有效旁白原样保留
  • 同框人物控制 — 2 人全清晰，3 人主次分离，4 人以上轮切
  • 情绪递进链 — 隐藏 → 泄露 → 对抗 → 爆发 → 收束
  • 七个反僵尸维度 — 视线/非对称/生理反馈/重心/环境互动/光影/表演指导
  • 生活化小动作词库 — 手部 / 整理动作 / 头部 / 身体 / 面部
  • 环境音效体系 — 分层音效 + 与 OS 隔离规则 + 动作触发音效
  • 质感关键词 — 实拍质感 + 35mm film + cinematic lighting + photorealistic
  • 固定约束词 — 字幕透明度 0% + 无背景音乐 + 丰富音效生动对白
